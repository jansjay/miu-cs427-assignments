  1. What is the difference between the following 2 statements?
    • setTimeout(booyah, 2000);
      Passes the function definition of booyah to the setTimeout function.
    • setTimeout(booyah(), 2000);
      Passes the output of the booyah() function to the setTimeout function.

  4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)
    Unobtrusive Javascript is "not including any javascript code in html code at all". Javascript code attaches any events during the load of javascript by manupulating the dom tree.

    If javascript is not supported, the html will be displayed without any issues.
    Also this makes it possible to maintain the separation of conserns during implementation. The data is defined in html, presentation using css and behavior using javascript
